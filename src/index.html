buton code:
$('#expandInbox').on('click', function() {
  $('#menu-inbox').bstreeCollapse('show');
});

$('#collapseInbox').on('click', function() {
  $('#menu-inbox').bstreeCollapse('hide');
});

Note: You could simply use Bootstraps collapse() method to expand/collapse the desired list group. But that would not automatically change the state icon.
